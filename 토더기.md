# 🤖 토더기 (Todeogi) - 김해시 AI 챗봇

김해시청 및 보건소 정보를 안내하는 RAG 기반 지능형 챗봇

## ✨ 특징

✅ **채팅 UI** - 직관적인 채팅 인터페이스 (Desktop + Mobile)
✅ **RAG 시스템** - 문서 기반 정확한 답변
✅ **웹 크롤링** - 김해시청 웹사이트 자동 수집
✅ **네이버 지도** - 보건소, 병원, 약국 위치 표시
✅ **재사용 가능 모듈** - 다른 프로젝트에서도 사용 가능

## 🚀 빠른 시작

### 1. 서버 설정

```bash
# 서버 디렉토리로 이동
cd server

# 패키지 설치
npm install

# 환경 변수 설정
copy .env.example .env
# .env 파일에 API 키 입력
```

### 2. 모듈 설치 (선택)

```bash
# 각 모듈의 의존성 설치 (재사용하려면)
cd modules/web-crawler && npm install
cd ../rag-system && npm install
cd ../naver-map && npm install
```

### 3. 서버 실행

```bash
cd server
npm start
```

서버가 http://localhost:3000 에서 실행됩니다!

### 4. 웹페이지 열기

- `todeogi_main.html` - 메인 페이지
- `todeogi_doctor.html` - 닥터 챗봇
- `todeogi_jumugaun.html` - 주무관 챗봇

## 🔑 API 키 발급

### OpenAI API 키
1. [OpenAI Platform](https://platform.openai.com/) 접속
2. API Keys 메뉴
3. Create new secret key
4. `.env`에 `OPENAI_API_KEY=sk-...` 입력

### 네이버 클라우드 API 키
1. [네이버 클라우드 콘솔](https://console.ncloud.com/) 접속
2. AI·NAVER API 메뉴
3. Application 등록 > Maps 선택
4. `.env`에 Client ID와 Secret 입력

## 📡 API 엔드포인트

### 1. 서버 상태 확인
```
GET /
```

### 2. RAG 기반 채팅
```
POST /api/chat
Body: { "message": "김해시청 위치가 어디인가요?" }
```

### 3. 웹 크롤링
```
POST /api/crawl

```

```
POST /api/rag/add-documents

```

```
GET /api/map/search?query=김해 보건소
```

### 6. 김해시 시설 검색
```
GET /api/map/gimhae/보건소
GET /api/map/gimhae/병원
GET /api/map/gimhae/약국
```

## 🔧 재사용 가능한 모듈

### 웹 크롤링 모듈
```javascript
const WebCrawler = require('./modules/web-crawler');

const crawler = new WebCrawler();
const data = await crawler.crawlStatic('https://example.com', ($) => {
    return { title: $('title').text() };
});
```

📖 [웹 크롤러 문서](modules/web-crawler/README.md)

### RAG 시스템 모듈
```javascript
const RAGSystem = require('./modules/rag-system');

const rag = new RAGSystem({ apiKey: 'your-key' });
await rag.addDocuments(['문서1', '문서2']);
const result = await rag.ask('질문');
```

📖 [RAG 시스템 문서](modules/rag-system/README.md)

### 네이버 지도 모듈
```javascript
const NaverMapAPI = require('./modules/naver-map');

const map = new NaverMapAPI({
    clientId: 'your-id',
    clientSecret: 'your-secret'
});
const places = await map.searchPlace('김해 보건소');
```

📖 [네이버 지도 문서](modules/naver-map/README.md)

## 📚 사용 예시

### 1. 김해시청 웹사이트 크롤링 → RAG 추가

```javascript
// 크롤링
const crawler = new WebCrawler();
const data = await crawler.crawlStatic('https://www.gimhae.go.kr', ($) => {
    return $('.content').text();
});

// RAG에 추가
const rag = new RAGSystem({ apiKey: process.env.OPENAI_API_KEY });
await rag.addDocuments([data]);

// 질문
const result = await rag.ask('김해시청 운영시간은?');
console.log(result.answer);
```

### 2. 김해시 보건소 찾기 → 지도 표시

```javascript
// 장소 검색
const naverMap = new NaverMapAPI({ ... });
const places = await naverMap.searchInGimhae('보건소');

// 프론트엔드에서 지도 표시
const mapWidget = new NaverMapWidget('map');
mapWidget.addMarkers(places);
```

## 🐛 문제 해결

### 서버가 시작되지 않을 때
```bash
# 포트 확인
netstat -ano | findstr :3000

# 다른 포트 사용
# .env 파일에서 PORT=3001로 변경
```

### CORS 오류
서버에서 CORS가 이미 설정되어 있습니다. 문제가 있다면 `server.js`의 CORS 설정을 확인하세요.

### API 키 오류
`.env` 파일이 `server/` 디렉토리에 있는지 확인하고, API 키에 따옴표가 없는지 확인하세요.
