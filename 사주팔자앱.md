# 사주팔자 Flutter 앱

AI가 분석하는 사주팔자 애플리케이션 (Flutter 버전)

React Native에서 Flutter로 완전히 마이그레이션된 프로젝트입니다.

## 주요 기능

- **대화형 입력**: AI와 자연스러운 대화를 통한 생년월일시 입력
- **사주팔자 계산**: 60갑자 기반 정확한 사주 계산
- **오행 분석**: 목화토금수 오행 분석
- **GPT-4 분석**: OpenAI GPT-4를 활용한 성격, 강점, 약점 분석
- **애니메이션**: 부드러운 UI 애니메이션과 전환 효과
- **Firebase 연동**: Firestore를 통한 사주 기록 저장

## 시작하기

### 1. Flutter 설치

먼저 Flutter SDK를 설치해야 합니다.

**Windows:**
```bash
# Chocolatey 사용
choco install flutter

# 또는 공식 사이트에서 다운로드
# https://flutter.dev/docs/get-started/install/windows
```

**macOS:**
```bash
# Homebrew 사용
brew install flutter
```

**Linux:**
```bash
# Snap 사용
sudo snap install flutter --classic
```

설치 확인:
```bash
flutter doctor
```

### 2. 프로젝트 설정

#### 2.1. 의존성 설치
```bash
cd SajupaljaFlutter
flutter pub get
```

#### 2.2. Firebase 설정

1. [Firebase Console](https://console.firebase.google.com/)에서 프로젝트 생성
2. Android/iOS 앱 등록
3. `google-services.json` (Android) 다운로드 → `android/app/` 폴더에 배치
4. `GoogleService-Info.plist` (iOS) 다운로드 → `ios/Runner/` 폴더에 배치
5. Firestore Database 생성 (테스트 모드로 시작)

#### 2.3. OpenAI API 키 설정

1. [OpenAI Platform](https://platform.openai.com/)에서 API 키 발급
2. `.env` 파일 수정:

```env
OPENAI_API_KEY=sk-your-actual-openai-api-key-here

# Firebase 설정 (필요시 - Flutter는 google-services.json 사용)
FIREBASE_API_KEY=your_firebase_api_key
FIREBASE_AUTH_DOMAIN=your_project.firebaseapp.com
FIREBASE_PROJECT_ID=your_project_id
FIREBASE_STORAGE_BUCKET=your_project.appspot.com
FIREBASE_MESSAGING_SENDER_ID=your_sender_id
FIREBASE_APP_ID=your_app_id
```

### 3. 실행

#### 개발 모드 실행
```bash
# Android 에뮬레이터 또는 연결된 기기
flutter run

# iOS 시뮬레이터 (macOS만)
flutter run -d ios

# Chrome 웹 브라우저 (개발용)
flutter run -d chrome
```

#### 디버그 빌드
```bash
# Android APK
flutter build apk --debug

# iOS (macOS만)
flutter build ios --debug
```



## 주요 화면 설명

### 1. InteractiveInputScreen (핵심!)
- AI와 대화하며 생년월일시를 단계별로 입력
- 8단계 입력 프로세스
- 각 입력마다 AI의 자연스러운 반응
- 애니메이션 효과와 진행도 표시

### 2. CalculatingScreen
- 사주팔자 계산 진행
- OpenAI GPT-4 API 호출
- 로딩 애니메이션과 순차적 메시지

### 3. ResultScreen
- 사주팔자 결과 표시
- 오행 분석
- GPT-4 성격 분석
- 강점/약점 표시
- 유료 상세 분석 유도

## 핵심 로직

### 사주 계산 (`saju_calculator.dart`)
- 60갑자 순환 계산
- 천간(天干) 10개, 지지(地支) 12개
- 년주, 월주, 일주, 시주 계산
- 오행(木火土金水) 분석

### AI 대화 (`narrative_scripts.dart`)
- 템플릿 기반 대화 스크립트
- 나이대, 계절, 시간대별 맞춤 응답
- 고풍스럽고 신비로운 어투

## 환경 변수

```env

FIREBASE_PROJECT_ID=...         # Firebase 프로젝트 ID (선택)
```

## 트러블슈팅

### Firebase 연결 오류
```bash
# Android
flutter clean
cd android && ./gradlew clean
cd .. && flutter pub get
flutter run

# iOS
cd ios
pod deintegrate
pod install
cd .. && flutter run
```

### OpenAI API 오류
- `.env` 파일의 API 키 확인
- API 사용량 및 결제 상태 확인
- `flutter pub get` 재실행

### 빌드 오류
```bash
flutter clean
flutter pub get
flutter pub upgrade
flutter run
```
