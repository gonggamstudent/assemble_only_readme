# 🌐 Enough - 올인원 커뮤니티 플랫폼

**"이것 하나로 충분하다"** - 모든 커뮤니티 플랫폼의 장점을 하나로 통합한 차세대 커뮤니티 사이트

## 📋 프로젝트 개요

Enough는 기존 커뮤니티 플랫폼들의 장점만을 선별하여 통합한 혁신적인 커뮤니티 플랫폼입니다. 디시인사이드의 자유로운 토론 문화, 루리웹의 정보 공유, 아카라이브의 커뮤니티 기능, 포스타입의 창작자 지원, 인스타그램의 시각적 매력, X(트위터)의 실시간성을 모두 담았습니다.

### 🎯 핵심 컨셉

- **하나로 충분** - 여러 플랫폼을 오갈 필요 없는 통합 경험
- **최고의 장점만** - 각 플랫폼의 검증된 기능들만 선별
- **크로스플랫폼** - 웹, 모바일 앱에서 동일한 경험

## ✨ 주요 기능

### 🗣️ 커뮤니티 (디시인사이드 + 루리웹)

- 갤러리 시스템 - 주제별 커뮤니티 생성 및 관리
- 실시간 댓글 시스템 - 빠른 소통과 토론
- 추천/비추천 시스템 - 커뮤니티 자체 정화
- 익명/반익명 시스템 - 자유로운 의견 표현

### 📱 소셜 미디어 (인스타그램 + X)

- 타임라인 피드 - 팔로우한 사용자들의 업데이트
- 스토리 기능 - 24시간 한정 콘텐츠
- 실시간 트렌딩 - 화제의 토픽 실시간 표시
- 해시태그 시스템 - 주제별 콘텐츠 발견

### 💰 창작자 지원 (포스타입)

- 후원 시스템 - 창작자 직접 후원
- 구독 모델 - 프리미엄 콘텐츠 구독
- 창작물 판매 - 디지털 콘텐츠 마켓플레이스
- 수익 분석 - 상세한 수익 통계

### 🎮 게임 & 엔터테인먼트 (아카라이브)

- 게임별 커뮤니티 - 게임 전용 공간
- 리뷰 시스템 - 게임, 영화, 드라마 리뷰
- 이벤트 시스템 - 커뮤니티 이벤트 개최
- 랭킹 시스템 - 활동 기반 등급제

## 🛠 기술 스택

### Frontend

- **Vue 3** + **TypeScript** - 타입 안전한 컴포넌트 기반 개발
- **Vite** - 초고속 개발 서버 및 빌드
- **Tailwind CSS** - 유틸리티 퍼스트 CSS 프레임워크
- **Pinia** - 상태 관리
- **Vue Router** - SPA 라우팅

### Mobile

- **Capacitor** - 크로스플랫폼 하이브리드 앱
- **PWA** - 프로그레시브 웹 앱 지원

### Backend (예정)

- **Node.js** + **Express** 또는 **Fastify**
- **PostgreSQL** - 메인 데이터베이스
- **Redis** - 캐싱 및 세션 관리
- **WebSocket** - 실시간 통신
- **AWS S3** - 미디어 파일 저장

## 🚀 시작하기

### 개발 환경 요구사항

- Node.js 22+
- Docker Desktop
- VSCode + Dev Containers 확장

### 프로젝트 설정

1. **저장소 클론**

   ```bash
   git clone <repository-url>
   cd enough
   ```

2. **Dev Container 실행**

   - VSCode로 프로젝트 열기
   - `Cmd + Shift + P` → "Dev Containers: Reopen in Container"

3. **Vue 프로젝트 생성**

   ```bash
   npm create vue@latest .
   # TypeScript, Router, ESLint, Prettier 선택
   ```

4. **의존성 설치**

   ```bash
   npm install
   ```

5. **Capacitor 설정**

   ```bash
   npm install @capacitor/core @capacitor/cli @capacitor/android @capacitor/ios
   npx cap init "Enough" "com.example.enough"
   ```

6. **개발 서버 실행**
   ```bash
   npm run dev
   # http://localhost:5173
   ```

## 🏗 프로젝트 구조

```
enough/
├── .devcontainer/          # Dev Container 설정
├── src/
│   ├── components/         # Vue 컴포넌트
│   │   ├── common/        # 공통 컴포넌트
│   │   ├── community/     # 커뮤니티 관련
│   │   ├── social/        # 소셜 미디어 관련
│   │   └── creator/       # 창작자 관련
│   ├── views/             # 페이지 컴포넌트
│   │   ├── Home.vue
│   │   ├── Community.vue
│   │   ├── Social.vue
│   │   └── Creator.vue
│   ├── stores/            # Pinia 스토어
│   ├── router/            # Vue Router 설정
│   ├── assets/            # 정적 자원
│   └── utils/             # 유틸리티 함수
├── android/               # Android 앱
├── ios/                   # iOS 앱 (macOS에서만)
├── dist/                  # 빌드 결과물
└── public/                # 공개 정적 파일
```

## 📱 플랫폼별 빌드

### 웹 배포

```bash
npm run build
# dist/ 폴더를 웹 서버에 배포
```

### Android 앱

```bash
npm run build
npx cap sync
npx cap open android
# Android Studio에서 APK 빌드
```

### iOS 앱 (macOS만)

```bash
npm run build
npx cap sync
npx cap open ios
# Xcode에서 IPA 빌드
```
