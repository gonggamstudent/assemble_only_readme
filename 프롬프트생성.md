# AI Prompt Generator

자연어를 AI 이미지/비디오 생성용 프롬프트로 변환하는 GUI 애플리케이션

Stable Diffusion, Midjourney 등의 AI 이미지 생성 도구에서 사용할 수 있는 고품질 프롬프트를 한국어로 쉽게 작성할 수 있습니다.

## 주요 기능

### 1. 번역 모드
한국어 요구사항을 영어로 번역합니다.

### 2. 프롬프트 모드
AI 이미지 생성에 최적화된 프롬프트로 변환합니다:
- 자동 가중치 할당 (중요도에 따라 1.1 ~ 1.5)
- Negative prompt 자동 생성
- 컨텍스트 분석을 통한 추가 태그 생성

## 설치

```bash
pip install -r requirements.txt
```

또는 직접 설치:
```bash
pip install deep-translator
```

## 실행

```bash
python main.py
```

## 사용 방법

1. 프로그램을 실행하면 GUI 창이 열립니다
2. 상단에서 **변환 모드**를 선택합니다:
   - **번역 모드**: 단순 한글 → 영어 번역
   - **프롬프트 모드**: AI 이미지 생성용 프롬프트 변환
3. 입력 영역에 한국어로 원하는 이미지를 설명합니다
4. **변환하기** 버튼을 클릭합니다
5. 결과가 출력 영역에 표시됩니다
6. **결과 복사** 버튼으로 클립보드에 복사할 수 있습니다

## 사용 예시

### 예시 1: 기본 사용
**입력:**
```
해당 이미지의 흰색머리를 가진 캐릭터가 화면 밖의 사용자를 인식하며
그들에게 반응하여 손으로 하트 제스처를 취하면서 웃도록 해야함.
```

**출력 (프롬프트 모드):**
```
(looking at viewer:1.4), (1girl:1.4), (white hair:1.3), (smile:1.3),
(heart hands gesture:1.3), (direct gaze:1.2), (interactive pose:1.2),
(positive emotion:1.1), (welcoming gesture:1.1), (engaging:1.1)

Negative prompt: worst quality, low quality, normal quality, lowres,
bad anatomy, bad hands, text, error, missing fingers, extra digit,
fewer digits, cropped, jpeg artifacts, signature, watermark, username,
blurry, looking away, back turned, profile view, looking down,
closed eyes, eyes closed, sad, angry, serious expression, frown
```

### 예시 2: 복잡한 장면
**입력:**
```
금발의 긴머리를 가진 소녀가 따뜻한 분위기에서 피스 사인을 하며 환하게 웃고 있다.
```

**출력:**
```
(1girl:1.4), (blonde hair:1.3), (smile:1.3), (bright smile:1.3),
(long hair:1.2), (peace sign:1.2), (warm atmosphere:1.2),
(positive emotion:1.1)

Negative prompt: worst quality, low quality, ...
```

## 프리셋 시스템

프롬프트 모드에서는 **프리셋**을 선택하여 기본 품질, 스타일, 조명 등을 미리 설정할 수 있습니다.

### 사용 가능한 프리셋 카테고리

#### 1. 화질 프리셋
- **기본**: masterpiece, best quality, highres
- **고품질**: masterpiece_portrait, high_quality, high_resolution
- **초고품질**: ultra detailed, 8k, extremely detailed
- **디테일 강조**: super detail, hyper detail, intricate_details
- **NSFW**: NSFW, masterpiece (성인 콘텐츠용)

#### 2. 화풍 프리셋
- **게임 CG**: game_cg, official art
- **미소녀**: bishoujo, cute, anime style
- **픽셀아트**: pixel_art, retro style
- **실사체**: realistic, photorealistic, photo
- **반실사체**: semi_realistic
- **일본풍 실사체**: manga_realistic, manga_photorealistic
- **미국풍 실사체**: cartoon_realistic, cartoon_photorealistic
- **유화**: oil painting, painterly
- **수채화**: watercolor, soft colors

#### 3. 조명 프리셋
- **자연광**: natural lighting, sunlight, soft lighting
- **극적인 조명**: dramatic lighting, cinematic lighting
- **측면 조명**: sidelighting, rim lighting
- **역광**: backlighting, backlit
- **네온**: neon lights, colorful lighting

#### 4. 구도 프리셋
- **초상화**: portrait, close-up, face focus
- **전신샷**: full body, full shot
- **역동적**: dynamic pose, action shot
- **시네마틱**: cinematic, movie still, film grain

### 프리셋 사용 예시

**입력:**
```
금발의 소녀가 사용자를 바라보며 웃고 있다.
```

**프리셋 선택:**
- 화질: 고품질
- 화풍: 게임 CG
- 조명: 극적인 조명

**출력:**
```
(masterpiece:1.5), (highres:1.5), (high_quality:1.5), (looking at viewer:1.4),
(1girl:1.4), (game_cg:1.3), (dramatic lighting:1.3), (blonde hair:1.3),
(smile:1.3), (direct gaze:1.2), (positive emotion:1.1)

Negative prompt: worst quality, low quality, lowres, bad anatomy, ...,
sketch, rough, flat lighting
```

### 프리셋 커스터마이징

`config.py`에서 프리셋을 추가하거나 수정할 수 있습니다:

```python
QUALITY_PRESETS = {
    "내 프리셋": {
        "positive": ["tag1", "tag2", "tag3"],
        "negative": ["unwanted1", "unwanted2"],
        "weight": 1.4  # 가중치 (1.0 ~ 1.5 권장)
    }
}
```

## 지원하는 키워드

### 외모
- 머리색: 흰색머리, 검은머리, 금발, 은발, 빨간머리, 파란머리, 초록머리, 보라머리, 분홍머리, 갈색머리
- 헤어스타일: 긴머리, 짧은머리, 트윈테일, 포니테일, 땋은머리, 웨이브, 곱슬머리

### 표정
- 웃음, 미소, 환하게 웃음, 윙크
- 행복, 슬픔, 화남, 놀람, 당황, 부끄러움

### 시선
- 사용자를 바라봄, 카메라를 바라봄, 시선, 눈 마주침
- 위를 바라봄, 아래를 바라봄, 옆을 바라봄

### 제스처/포즈
- 하트, 피스, 손 흔들기, 엄지척, 가리키기
- 서있는, 앉아있는, 누워있는, 달리는, 걷는, 점프

### 의상
- 교복, 원피스, 치마, 정장, 수영복, 후드티

### 분위기/조명
- 따뜻한, 차가운, 부드러운 조명, 밝은, 어두운
- 햇빛, 석양, 밤, 실내, 실외

### 카메라 앵글
- 클로즈업, 전신, 상반신, 위에서, 아래에서

## 키워드 추가/수정

자신만의 키워드를 추가하여 변환 품질을 높일 수 있습니다.

**상세 가이드**: `KEYWORD_GUIDE.md` 참고

**빠른 사용법**:
1. `config.py` 열기
2. `KEYWORD_MAP`에 추가:
   ```python
   "한국어키워드": ("english_tag", 1.3),
   ```
3. 저장 후 프로그램 재시작
