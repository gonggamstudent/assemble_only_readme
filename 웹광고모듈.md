# 🎯 범용 광고 관리 시스템

이 시스템은 모든 웹페이지에서 사용 가능한 독립적인 광고 관리 솔루션입니다.

## 🌟 주요 기능

### ✅ 완전 독립적인 모듈
- 웹페이지에서 분리 가능한 광고 관리 시스템
- localStorage를 통한 데이터 영속성
- React 환경과 순수 HTML 환경 모두 지원

### ✅ 실시간 광고 관리
- 광고 추가/수정/삭제 즉시 반영
- 슬롯별 활성화/비활성화
- 설정 내보내기/가져오기 (백업/복원)

### ✅ 다양한 광고 형태 지원
- **이미지 광고**: JPG, PNG, WebP 등
- **동영상 광고**: MP4, WebM (자동재생/음소거)
- **GIF 광고**: 애니메이션 이미지
- **텍스트 광고**: 미디어 없는 순수 텍스트

### ✅ 유연한 배치 시스템
- **Fixed**: 화면에 고정 (상단/하단/사이드바)
- **Absolute**: 특정 위치에 절대 배치
- **Static**: 컨텐츠 사이 자연스러운 배치

## 📋 사용 방법

### 1. React 환경에서 사용

```jsx
import UniversalAdManager from './modules/UniversalAdManager'
import AdSlot from './components/AdSlot'

function MyApp() {
  const [showManager, setShowManager] = useState(false)
  
  return (
    <div>
      {/* 광고 관리 패널 */}
      <UniversalAdManager 
        isVisible={showManager}
        onToggle={() => setShowManager(!showManager)}
        position={{ top: '20px', right: '20px' }}
        theme="light"
      />
      
      {/* 광고 슬롯 배치 */}
      <AdSlot slotId="topBanner" />
      <AdSlot slotId="rightSidebar" />
      <AdSlot slotId="bottomBanner" />
      
      {/* 여기에 일반 컨텐츠 */}
      <div>일반 웹페이지 내용</div>
      
      <AdSlot slotId="inlineAd1" />
    </div>
  )
}
```

### 2. 순수 HTML 환경에서 사용

```html
<!DOCTYPE html>
<html>
<head>
    <title>광고 시스템 테스트</title>
</head>
<body>
    <h1>내 웹페이지</h1>
    <p>일반 컨텐츠...</p>
    
    <!-- 광고 시스템 로더 스크립트 -->
    <script src="./utils/AdSystemLoader.js"></script>
    
    <script>
        // 광고 시스템 로드
        loadAdSystem({
            enableManager: true,
            managerPosition: { top: '20px', right: '20px' },
            autoLoadSlots: ['topBanner', 'rightSidebar'],
            theme: 'light'
        }).then(result => {
            if (result.success) {
                console.log('광고 시스템 로드 완료!')
            }
        })
    </script>
</body>
</html>
```

## 🎛️ 광고 슬롯 설정

### 기본 제공 슬롯

1. **topBanner**: 상단 배너 (80px 높이)
2. **rightSidebar**: 우측 사이드바 (200x300px)
3. **bottomBanner**: 하단 배너 (60px 높이)
4. **inlineAd1**: 인라인 광고 (컨텐츠 사이)
5. **floatingAd**: 플로팅 광고 (애니메이션 효과)

### 커스텀 슬롯 추가


## 🔧 관리 패널 사용법

### 1. 광고 추가
1. 우상단 "🎯 광고 관리" 버튼 클릭
2. 원하는 슬롯 선택
3. 광고 정보 입력:
   - 제목, 내용
   - 미디어 타입 선택
   - URL, 링크, 색상 설정
4. "➕ 광고 추가" 클릭

### 2. 광고 삭제
1. 슬롯 선택 → 광고 선택
2. "🗑️ 선택된 광고 삭제" 클릭

### 3. 슬롯 제어
- **ON/OFF**: 슬롯 전체 활성화/비활성화
- **선택**: 관리할 슬롯 선택

### 4. 데이터 백업/복원
- **📤 설정 내보내기**: JSON 파일로 백업
- **📥 설정 가져오기**: 백업 파일 복원
- **🔄 초기화**: 모든 설정 초기화

## ❓ FAQ

**Q: 광고가 표시되지 않아요.**
A: 슬롯이 활성화되어 있고, 광고 데이터가 있는지 확인하세요.

**Q: 모바일에서 일부 광고가 보이지 않아요.**
A: CSS에서 반응형 설정을 확인하세요. 일부 슬롯은 모바일에서 숨겨집니다.

**Q: 데이터가 사라졌어요.**
A: 브라우저 캐시 정리 시 localStorage가 삭제될 수 있습니다. 정기적으로 백업하세요.
